PORTFOLIO

Master's Degree Thesis - Secure Decentralized IoT Infrastructure

Aims to showcase an alternative concept for IoT Infrastructures, which can help in designing a fault-tolerant and scalable infrastructure. The main objective consists of building a dynamic and self-configurable infrastructure on top of a structured Peer-to-Peer network. Moreover, a set of protocols must be provided for embracing the most varied devices, as well as data access, streaming and persistence. Thus, it is essential to have mechanisms that guarantee the privacy and security of the information.

Services Engineering Group Project - Virtual Study Room

This project consists of an online adaptative study environment for students, The users could easily find a table, where other people are studying the same topics, as well as found valuable resources. It was developed a Real-time web application based on a Service Oriented Architecture. Its main features are instant messaging, VoIP, Screen Sharing and File-sharing. More info:

Informatics Engineering Graduation Project - YouInteract

This project resides in a system that could, simultaneously, entertain and be useful to a common person that is passing by the hall of a university department, tourist office, hospital, among others. It was developed an interacting platform application, based on a Plugin Architecture, in order to allow other students to develop and deploy applications easily as plugins. The users interact with the platform through a Microsoft Kinect. This platform is currently running at Eletronics, Telecommunication and Informatics Department of Aveiro Univeristy.
